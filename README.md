# TOEIC Vocabulary Test (托业词汇模拟测试)

一个基于 Web 的现代化托业词汇练习系统，采用 **玻璃拟态 (Glassmorphism)** 设计风格，旨在提供沉浸式的学习体验。

## 🌟 核心功能

-   **丰富题库**：内置 260 道精心筛选的托业核心词汇题，涵盖四大商务场景。
-   **职场分类**：
    -   💼 **商务办公 (Office)**: 60 题 (涵盖协商、招聘、流程管理等)
    -   📊 **金融财务 (Finance)**: 60 题 (涵盖审计、投资、资产管理等)
    -   ✈️ **差旅接待 (Travel)**: 60 题 (涵盖行程、海关、酒店设施等)
    -   💻 **技术协作 (Tech)**: 80 题 (涵盖架构、安全、系统优化等)
-   **实时反馈**：每道题点击后立即显示正确答案与**中文深度解析**。
-   **倒计时器**：内置 15 秒限时挑战机制，模拟真实考场压力。
-   **响应式设计**：完美适配不同屏幕尺寸，提供流畅的横向布局体验。

## 🛠️ 技术实现

-   **HTML5 & CSS3**: 使用 CSS 变量和 Flexbox/Grid 构建现代化玻璃效果 UI。
-   **Vanilla JavaScript (ES6+)**: 原生 JS 实现题目随机抽样、答题逻辑、计时器控制。
-   **轻量化**: 无需任何第三方库或依赖，单页面应用极致流畅。

## 🚀 快速开始

只需在浏览器中打开项目的 `index.html` 文件即可开始练习。

```bash
# 克隆/下载项目后，双击 index.html 即可
```

## 📈 练习规则

1.  从主屏幕选择一个业务场景。
2.  每轮测试包含 **10 道** 随机题目。
3.  每题限时 **15 秒**。
4.  答题结束后可查看最终得分及评价。
