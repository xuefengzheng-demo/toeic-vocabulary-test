# TOEIC Vocabulary Test (托业词汇模拟测试)

一个基于 Web 的现代化托业词汇练习系统，采用 **前后端分离** 架构与 **玻璃拟态 (Glassmorphism)** 设计风格，旨在提供沉浸式的学习体验。

## 🌟 核心功能

-   **丰富题库**：内置 260 道精心筛选的托业核心词汇题，涵盖四大商务场景。
-   **职场分类**：
    -   💼 **商务办公 (Office)**: 70 题
    -   📊 **金融财务 (Finance)**: 70 题
    -   ✈️ **差旅接待 (Travel)**: 70 题
    -   💻 **技术协作 (Tech)**: 90 题
-   **实时反馈**：每道题点击后立即显示正确答案与**中文深度解析**。
-   **倒计时器**：内置 15 秒限时挑战机制，模拟真实考场压力。
-   **响应式设计**：完美适配不同屏幕尺寸，提供流畅的交互体验。

## 🏗️ 技术架构

项目采用前后端分离模式：
-   **前端 (本项目)**: 原生 HTML5, CSS3 (Vanilla), JavaScript (ES6+)。
-   **后端 (API)**: Node.js + Express + JSON 存储。

## 🚀 快速开始

### 1. 启动后端 API
本项目依赖 [toeic-api](file:///d:/Antigravity/toeic-api) 提供题目数据。请确保后端服务已在 `localhost:3000` 运行。

```bash
# 进入后端目录
cd ../toeic-api
# 安装依赖
npm install
# 启动服务器
npm start
```

### 2. 启动前端
在浏览器中打开项目的 `index.html` 文件即可开始练习。

```bash
# 双击 index.html 或使用 Live Server 插件打开
```

## 📈 练习规则

1.  从主屏幕选择一个业务场景。
2.  每轮测试包含 **10 道** 随机题目。
3.  每题限时 **15 秒**。
4.  答题结束后可查看最终得分及评价。
